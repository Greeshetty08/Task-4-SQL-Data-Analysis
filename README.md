# Task 4 - SQL for Data Analysis

## Objective
Use SQL queries to extract, filter, join, aggregate, and analyze data from a structured database.

## Dataset
- Chinook SQLite Database

## Tool Used
- DB Browser for SQLite

## SQL Concepts Covered
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate functions: SUM, AVG
- INNER JOIN
- LEFT JOIN
- Subqueries
- Views
- Indexes

## Queries Performed
1. Displayed customer records using SELECT.
2. Filtered customers by country using WHERE.
3. Sorted customer names using ORDER BY.
4. Calculated total spending by customer using GROUP BY and SUM.
5. Calculated average invoice value using AVG.
6. Joined customers and invoices using INNER JOIN.
7. Used LEFT JOIN to display customer invoice details.
8. Used a subquery to find invoices above average value.
9. Created a view for customer spending analysis.
10. Created an index to optimize query performance.

## Outcome
This task helped demonstrate SQL skills for data analysis, including querying relational databases, summarizing business data, and improving query performance.
